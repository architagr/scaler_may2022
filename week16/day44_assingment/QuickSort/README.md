# Problem Description

Given an integer array **A**, sort the array using **QuickSort.**

###### Problem Constraints

```
1 <= |A| <= 10^5

1 <= A[i] <= 10^9
```

###### input format

``` 
First argument is an integer array A.
```

###### Output Format

```
Return the sorted array.
```

###### Example Input

```
# input 1: 
    A = [1, 4, 10, 2, 1, 5]
# output 1: 
    [1, 1, 2, 4, 5, 10]
# input 2: 
    A = [3, 7, 1]
# output 2: 
    [1, 3, 7]
```
