# Problem Description

Given an array, A of integers of size N. Find the maximum value of j - i such that A[i] <= A[j].

###### Problem Constraints

```
1 <= N <= 10^5

-10^9 <= A[i] <= 10^9
```

###### input format

``` 
First argument is an integer array A of size N.
```

###### Output Format

```
Return an integer denoting the maximum value of j - i.
```

###### Example Input

```
# input 1: 
     A = [3, 5, 4, 2]
# output 1: 
    2
```
