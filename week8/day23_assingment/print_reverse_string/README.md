# Problem Description

Write a recursive function that, given a string S, prints the characters of S in reverse order.

###### Problem Constraints

```
1 <= |s| <= 1000
```

###### input format

``` 
First line of input contains a string S.
```

###### Output Format

```
Print the character of the string S in reverse order.
```

###### Example Input

```
# input 1 : 
    scaleracademy
# output 1: 
    ymedacarelacs
# input 2 : 
    cool
# output 2: 
    looc
```
