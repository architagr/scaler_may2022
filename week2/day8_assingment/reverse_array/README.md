# Problem Description

You are given a constant array **A**.

You are required to return another array which is the reversed form of the input array.

###### Problem Constraints

```
1 <= A.size() <= 10000

1 <= A[i] <= 10000
```

###### input format

``` 
First argument is a constant array A.
```

###### Output Format

```
Return an integer array.

```

###### Example Input

```
# input 1 : 
    A = [1,2,3,2,1]
# output 1: 
    [1,2,3,2,1] 
# input 2 : 
    A = [1,1,10]
# output 2: 
    [10,1,1]
```
