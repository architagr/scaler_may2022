# Problem Description

Given a sequence f(A) = f(A-1) + f(A-2) + f(A-3) + A . Calculate the Ath term of the sequence.

Given f(0)=1; f(1)=1; f(2)=2;

###### Problem Constraints

```
0<=n<=20
```

###### input format

``` 
First and only argument is an integer A.
```

###### Output Format

```
Return an integer denoting the Ath term of the sequence.
```

###### Example Input

```
# input 1: 
    3
# output 1: 
    2
# input 2: 
    7
# output 2: 
    2
```
