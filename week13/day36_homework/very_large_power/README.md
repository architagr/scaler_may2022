# Problem Description

Given two Integers A, B. You have to calculate (A ^ (B!)) % (1e9 + 7).

"^" means power,

"%" means "mod", and

"!" means factorial.

###### Problem Constraints

```
1 <= A, B <= 5e5
```

###### input format

``` 
First argument is the integer A

Second argument is the integer B
```

###### Output Format

```
Return one integer, the answer to the problem
```

###### Example Input

```
# input 1 : 
    A = 1
    B = 1
# output 1: 
    1
# input 2 : 
    A = 2
    B = 2
# output 2: 
    4
```
