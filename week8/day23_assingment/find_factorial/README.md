# Problem Description

Write a program to find the factorial of the given number A.

###### Problem Constraints

```
0 <= A <= 12
```

###### input format

``` 
First and only argument is an integer A.
```

###### Output Format

```
Return an integer denoting the factorial of the number A.
```

###### Example Input

```
# input 1 : 
    A = 4
# output 1: 
    24
# input 2 : 
    A = 5
# output 2: 
    120
```
